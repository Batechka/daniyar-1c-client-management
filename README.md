# 📦 Учет клиентов (1С:Предприятие 8.3)

## 📌 Описание проекта
Система автоматизации учета клиентов организации в 1С:Предприятие 8.3.

Позволяет:
- хранить данные клиентов,
- учитывать взаимодействия с ними,
- регистрировать услуги/заказы,
- анализировать продажи и активность клиентов.

---

## ⚙️ Используемые технологии
- 1С:Предприятие 8.3
- Справочники, Документы, Регистры
- Табличные части
- СКД (Система Компоновки Данных)
- Обработки и отчеты

---

## 📁 Структура проекта
### Справочники:
- **Клиенты**: ФИО, Телефон, Email, Адрес, Дата регистрации
- **Услуги**: Наименование, Цена
- **Сотрудники**: ФИО, Должность, Телефон

### Документы:
- **ЗаказКлиента**: хранит заказы, суммирует услуги
- **ОбращениеКлиента**: фиксирует обращения (звонки, визиты, письма)

### Регистры:
- **Продажи** (накопительный): хранит продажи по клиентам и услугам
- **ИсторияОбращений**: хранит обращения клиентов

### Отчеты:
- **Отчет по клиентам**: список клиентов, количество заказов, сумма покупок
- **Отчет по продажам**: услуги, количество продаж, общий доход

### Формы:
- Список клиентов
- Заказ
- Обращение
- Кнопки: “Создать заказ”, “Добавить обращение”
